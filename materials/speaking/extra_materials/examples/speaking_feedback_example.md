# speaking_feedback_example.md

## Input

Part 2 topic: Describe a useful skill you learned.
Candidate transcript: self-created practice transcript.

I want to talk about cooking. I learned it last year because I live alone. At first I only cook noodles, but later I watched videos and tried some dishes. It is useful because I can save money and eat healthy food. Sometimes I make mistakes, like the food is too salty, but now it is better. I think everyone should learn cooking because it is very practical.

## Estimated band

Estimated Speaking Band: 6.0
Confidence: low to medium because pronunciation cannot be fully judged from text.

## Criterion diagnosis

- Fluency and Coherence: Response is coherent and easy to follow, but extension is limited for a 1-2 minute long turn.
- Lexical Resource: Vocabulary is adequate for a familiar topic, but range is narrow.
- Grammatical Range and Accuracy: Mostly simple structures; one tense issue: `I only cook noodles` should be `I only cooked noodles`.
- Pronunciation: Cannot be reliably assessed from transcript. Ask for audio or mark as not assessed.

## Skill output rule

When only a transcript is provided, do not invent detailed pronunciation feedback. Use: `Pronunciation cannot be reliably scored without audio; only possible risks are noted if the transcript indicates repeated hesitation or word stress markers.`

## Rewrite direction

Extend with a specific moment, a problem solved, and a comparison before/after learning the skill.
