# IELTS Examiner Materials

本目录用于承载 `ielts-examiner` 的可扩展资料，不再把“个人备考资料”单独列为第二类。所有后续补充材料统一按听说读写归档。

## 目录规则

每个科目都有两个固定子目录：

- `extra_materials/`：官方资料、半官方资料、第三方讲义、taxonomy、样例反馈等可复用资料。
- `user_practice_drafts/`：用户自己的练习原稿、错题记录、二稿、口语转写、听力/阅读错题表。

## 科目目录

| 科目 | 路径 | 可放内容 |
|------|------|----------|
| Listening | `listening/` | 官方听力样题、raw score 表、听力错题表、拼写/单复数/词数限制归因 |
| Speaking | `speaking/` | 官方口语评分标准、口语文字稿、录音转写、Part 1/2/3 反馈样例 |
| Reading | `reading/` | Academic/GT 阅读样题、raw score 表、TF/NG、matching headings、summary completion 错题 |
| Writing | `writing/` | 官方写作评分标准、Task 1/Task 2 样题、作文原稿、批改稿、二稿复盘 |

## 使用边界

- `extra_materials/` 可作为 skill 的长期参考资料。
- `user_practice_drafts/` 默认只存用户主动放入的练习材料，不自动写入聊天内容。
- 若第三方资料与官方标准冲突，以 `references/official_sources.md` 和官方 PDF 为准。
- 受版权保护的整本教材或真题集不要放入本目录；只保留必要摘录、来源记录或用户自己的练习产出。
