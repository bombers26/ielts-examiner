# Listening Index

## Use This Directory When

用户询问 Listening raw score、听力错题、拼写/单复数/词数限制错误、数字日期听写问题时，优先读取本索引。

- `extra_materials/official/ielts-listening-sample-tasks-2023.pdf`

## Taxonomy

- `extra_materials/taxonomy/scoring_tables.md`
  - 用于 raw score anchor-based estimate。
  - 不要扩展为官方固定完整换算表。
- `extra_materials/taxonomy/error_taxonomy.md`
  - 用于归因拼写、单复数、词数限制、听出但写错、未听出等错误。

## User Practice Drafts

放入 `user_practice_drafts/`：

- raw score 记录
- 错题表
- 原题限制，如 `NO MORE THAN TWO WORDS AND/OR A NUMBER`
- 用户错误答案与正确答案

## Output Rule

先说明 raw score 估分的不确定性，再归因错题类型，最后给一个 10-30 分钟训练任务。
