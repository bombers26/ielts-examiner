# IELTS skill sources 2026-05-14

用途：为后续 IELTS 批改/训练 skill 建立本地资料底座。

## 目录结构

- `official/`: 已下载或尝试下载的官方 PDF，含 `download_manifest*.json` 与 SHA256。
- `taxonomy/`: 可直接放进 skill `references/` 的评分、题型、错误、词汇 taxonomy。
- `examples/`: 可直接放进 skill `examples/` 的批改与错题反馈样例。
- `source_log.md`: 本次核验来源、用途、下载状态和版权边界。

## 使用边界

1. 官方/半官方资料是评分依据；机构资料只能作表达训练或课堂活动参考。
2. Cambridge IELTS 真题集不得整本搬入 skill；只抽象题型规则、答案解释模式和常见陷阱。
3. 真实考试题、整篇版权范文、音频转录全文不放入 skill；skill 中只保留自造样例、短摘录索引和来源链接。
4. 评分输出必须明确“不等同于官方 IELTS examiner score”，并用四项标准解释估分。

## 快速入口

- 评分换算：`taxonomy/scoring_tables.md`
- Task 1：`taxonomy/task1_patterns.md`
- Task 2：`taxonomy/task2_question_types.md`
- 错误 taxonomy：`taxonomy/error_taxonomy.md`
- 词汇库：`taxonomy/vocabulary_bank.md`
- 写作批改示例：`examples/task1_feedback_example.md`, `examples/task2_feedback_example.md`
- 口语反馈示例：`examples/speaking_feedback_example.md`
- 阅读/听力错题表：`examples/reading_error_table_example.md`, `examples/listening_error_table_example.md`
