# vocabulary_bank.md

原则：优先准确、自然、可控；不要把“高级”理解为生僻。

## Task 1 trends

| 简单表达 | 可控升级 | 用法 |
|---|---|---|
| go up | increase / rise / climb | 中性上升 |
| go down | decrease / decline / fall | 中性下降 |
| go up quickly | rise sharply / increase markedly | 快速显著上升 |
| go down slowly | decline gradually | 缓慢下降 |
| stay the same | remain stable / remain relatively unchanged | 稳定 |
| change a lot | fluctuate considerably | 波动 |
| the most | the highest figure / the largest share | 极值 |
| more than | exceed / be higher than | 比较 |
| twice as much as | be twice as high as / roughly double | 倍数 |
| become bigger | the gap widened | 差距变化 |

## Task 1 maps/processes

- was replaced by / was converted into / was expanded / was demolished
- to the north of / adjacent to / opposite / in the centre of
- first / subsequently / at the next stage / finally
- raw materials / undergo / be processed / be transported / be packaged

## Task 2 argument language

| 功能 | 表达 |
|---|---|
| 表明立场 | I agree with this view to a large extent. |
| 让步 | While this concern is valid, it does not fully account for... |
| 因果 | This can lead to / This is largely because... |
| 举例 | A clear example is... |
| 限定 | in many cases / for some groups / in the long term |
| 总结 | Overall, the stronger argument is that... |

## 禁用/慎用

- `nowadays` 开头套句过多时会显得模板化。
- `with the development of society` 信息密度低。
- 不确定搭配的生僻词不要使用；LR 评分重视准确和适切。
