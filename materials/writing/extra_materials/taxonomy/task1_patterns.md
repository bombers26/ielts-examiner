# task1_patterns.md

适用：IELTS Academic Writing Task 1。
核心评分轴：Task Achievement, Coherence & Cohesion, Lexical Resource, Grammatical Range & Accuracy。

## 通用结构

1. Introduction: 改写题目，不解释原因。
2. Overview: 2 个最显著总体特征；优先最高/最低、整体趋势、主要阶段、关键变化。
3. Body 1: 第一组可比较信息。
4. Body 2: 第二组可比较信息。

## 图表题型

| 类型 | 必须识别 | 常见失分 |
|---|---|---|
| line graph | 起点/终点、上升下降、峰值低点、交叉点 | 逐点流水账；没有 overview；趋势词误用 |
| bar chart | 组间最高最低、显著差距、类别排序 | 每根柱子都写；比较不足 |
| pie chart | 最大/最小占比、相邻类别、时间变化下的份额转移 | 只列百分比；没有合并小类别 |
| table | 行列维度、极值、异常值、可分组模式 | 顺序抄表；缺少筛选 |
| map | before/after、拆除/新增/改建、方位关系 | 写成路线说明；方位词混乱 |
| process | 起点终点、阶段数、线性/循环、人工/自然过程 | 加个人评价；遗漏关键阶段 |
| mixed charts | 两图共同主题、单位差异、互相解释关系 | 两图分开复述，缺少综合比较 |

## Overview checklist

- 至少包含 2 个总体特征。
- 不写具体细碎数字，除非该数字本身是核心极值。
- 不写原因、预测、个人观点。
- process/map 也必须有 overview：阶段/布局变化的总体概括。

## Cambridge IELTS 真题集抽象规则

- 真题集的价值在于“任务边界”和“答案解释方式”，不在于背范文。
- 对每道题抽象：图表类型、数据单位、比较维度、overview 目标、body 分组方式。
- 不将题干、图表、完整范文整篇复制进 skill。
