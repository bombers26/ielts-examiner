# institution_handout_alignment.md

用途：评估机构讲义是否与官方 IELTS 口径一致。

## Alignment checklist

| 讲义主张 | 官方对齐判断 | 处理 |
|---|---|---|
| Task 1 必须有 overview | aligned | 保留，标为 high priority |
| Task 1 可以写原因分析 | conflict | 删除或标注为错误；Task 1 描述信息，不做主观原因分析 |
| Task 2 一定要背万能模板 | conflict/risky | 改成可迁移逻辑框架，避免 memorised language |
| Reading TFNG 中 False 等于没提到 | conflict | 纠正：False 需要文本反证，Not Given 才是没信息 |
| Listening 拼写错误扣分 | aligned | 保留，加入 spelling log |
| Speaking 必须使用高级词汇才高分 | misleading | 改成 precise and appropriate lexical resource |

## 输出格式

1. `aligned`: 可以直接纳入 skill。
2. `partly aligned`: 需要限定条件。
3. `conflict`: 与官方口径冲突，不纳入。
4. `unsupported`: 没有官方依据，可作为个人策略但不能当评分规则。
