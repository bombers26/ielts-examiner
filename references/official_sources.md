# Official IELTS Sources Reference

以下为本 skill 引用的官方来源和本地资料。

## 官方 IELTS 来源

### 评分标准

1. **IELTS Writing Band Descriptors** (官方 PDF)
   - URL: https://ielts.org/cdn/ielts-guides/ielts-writing-band-descriptors.pdf
   - 用途: Writing Task 1 (TA/CC/LR/GRA) 和 Task 2 (TR/CC/LR/GRA) 评分标准
   - 本地位置: `materials/writing/extra_materials/official/ielts-writing-band-descriptors.pdf`

2. **IELTS Speaking Band Descriptors** (官方 PDF)
   - URL: https://ielts.org/cdn/ielts-guides/ielts-speaking-band-descriptors.pdf
   - 用途: Speaking 四项评分标准 (FC/LR/GRA/P)
   - 本地位置: `materials/speaking/extra_materials/official/ielts-speaking-band-descriptors.pdf`

3. **IELTS Scoring in Detail**
   - URL: https://ielts.org/take-a-test/your-results/ielts-scoring-in-detail
   - 用途: Overall band score 计算方式、Listening/Reading raw score → band 对照表（平均参考值）
   - 获取日期: 2026-05-14

4. **IELTS Academic Format — Listening**
   - URL: https://ielts.org/take-a-test/test-types/ielts-academic-test/ielts-academic-format-listening
   - 用途: Listening 4 parts × 10 题格式、题型列表（multiple choice, matching, labelling, completion, short answer）

### 官方样题与备考资源

5. **IELTS Academic Writing Sample Tasks 2023** (官方 PDF)
   - URL: https://ielts.org/cdn/Sample-tests/ielts-academic-writing-sample-tasks-2023.pdf
   - 本地位置: `materials/writing/extra_materials/official/ielts-academic-writing-sample-tasks-2023.pdf`

6. **British Council — IELTS practice Academic Writing Task 1**
   - URL: https://takeielts.britishcouncil.org/take-ielts/prepare/free-ielts-english-practice-tests/writing/academic/task-1
   - 用途: 官方练习题和 model answer 下载入口

7. **British Council — Academic Writing Task 1: Grammatical Structures for Describing Diagrams**
   - URL: https://takeielts.britishcouncil.org/sites/default/files/academic_writing_task_1_grammatical_structures_for_describing_diagrams_0.pdf
   - 用途: diagram/process 语法结构

## 本地分科资料目录

额外资料与用户练习原稿统一放入 `materials/`，按听说读写分科：

- Listening: `materials/listening/extra_materials/` 与 `materials/listening/user_practice_drafts/`
- Speaking: `materials/speaking/extra_materials/` 与 `materials/speaking/user_practice_drafts/`
- Reading: `materials/reading/extra_materials/` 与 `materials/reading/user_practice_drafts/`
- Writing: `materials/writing/extra_materials/` 与 `materials/writing/user_practice_drafts/`

## 使用协议

- 所有官方来源仅用于备考辅助，不用于商业用途
- 评分预估仅供参考，不替代官方 IELTS 成绩
- 本地资料整理日期: 2026-05-13
- 作者应定期核对官方 URL 是否有效
