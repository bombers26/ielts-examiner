# Student Material Intake Policy

## 1. Purpose

定义本 skill 如何处理用户上传的自有雅思资料（非本 skill 内置资料），确保处理过程一致、可追溯、不产生误导。

## 2. Trigger Phrases

以下用户意图触发资料导入模式：

| 触发短语 | 匹配关键词 |
|----------|------------|
| "我上传了雅思资料" | 上传 / 资料 / 分享 |
| "根据这份机构讲义训练我" | 讲义 / 训练 / 机构 |
| "帮我整理这份雅思笔记" | 整理 / 笔记 / 总结 |
| "根据我的错题做诊断" | 错题 / 诊断 / 分析 |
| "根据这篇作文/口语稿给反馈" | 作文 / 口语稿 / 反馈 |
| 隐含触发：用户粘贴了大段英文并请求"根据这个来训练" | 训练 / 学习 / 练习 |

## 3. Material Intake Workflow

### Step 1: Identify Material Type

识别用户上传内容的类型：

| 类型 | 特征 |
|------|------|
| **机构讲义** | 结构化、有标题层级、含"考点""技巧""模板"等词汇 |
| **个人笔记** | 非正式格式、缩写、中英混杂、含个人标注 |
| **真题回忆** | 含题目描述、无完整原文 |
| **错题集** | 以表格或列表呈现、有错误答案和正确答案对比 |
| **写作练习** | 完整作文、含或不含题目 |
| **口语稿** | 对话式文字、含 Part 1/2/3 标注 |
| **备考计划** | 日程表、目标分数、学习时长分配 |
| **机构宣传材料** | 含"保分""提分 X 天"等营销用语 |

### Step 2: Source Credibility Assessment

| 等级 | 标准 | 行为 |
|------|------|------|
| **T1 官方** | 来自 ielts.org / British Council / IDP / Cambridge 官网 | 可作为核心依据 |
| **T2 半官方** | Cambridge 真题集、British Council 授权出版物 | 可参考，标注来源 |
| **T3 第三方** | 知名培训机构（新东方、环球雅思等）、高分考生分享 | 提取可用规则，标注潜在偏差 |
| **T4 自制** | 用户个人笔记、错题集 | 作为训练线索使用，不引用为"标准" |
| **T0 未知** | 无法识别来源 | 标注"来源不可验证" |

默认不带来源标注的上传内容归类为 T3-T4 之间，除非有明显的官方来源特征。

### Step 3: Extract Actionable Rules

从资料中提取可复用的规则/技巧：

**提取原则：**
- 优先提取与官方评分标准直接相关的规则（如 overview 写法、段落结构）
- 标注规则与官方标准的一致性程度（一致 / 不完全一致 / 冲突）
- 对模板化表达进行标注："这句在官方标准中被视为机械套用"
- 忽略营销型内容（"保过""押题命中率 90%"）

### Step 4: Align with Official IELTS Standards

将提取的规则与内置官方标准对照：

- **一致**：确认用户资料与官方标准不矛盾，推荐保留使用
- **部分一致**：指出哪些部分准确、哪些部分有偏差，建议修正
- **冲突**：明确指出冲突原因，引用官方标准原文，建议以官方为准

### Step 5: Output Personalised Recommendations

输出格式：

```
## 资料分析报告

**来源可信度：** [T1/T2/T3/T4/T0]

### 提取的可复用规则
1. [规则 1] — ✅ 与官方一致
2. [规则 2] — ⚠️ 部分偏差（说明）
3. [规则 3] — ❌ 与官方冲突（说明 + 官方标准引用）

### 个人化训练建议
根据你的资料，推荐以下训练重点：
- [建议 1]
- [建议 2]

### 冲突摘要
- [冲突项]：你的资料说 X，官方标准说 Y，建议以官方为准
```

## 4. Conflict Resolution Rules

1. **T1 > 一切**：任何来源与官方标准冲突时，以官方标准为准
2. **T2 参考但不绝对**：T2 材料中的方法学建议可以作为练习参考，但不作为评分标准
3. **T3 谨慎使用**：T3 材料可能包含简化的"速成技巧"，这些技巧在低分段可能有效，在高分段可能成为扣分点
4. **T4 不推广**：自制笔记中的个人策略不保证对他人有效，不主动推荐
5. **T0 标注风险**：来源不明的材料标注使用风险

## 5. Privacy & Data Handling

- 用户上传内容在当前对话中处理
- 不写入 skill 目录下的任何文件
- 不发送到外部服务
- 聊天历史由当前宿主管理，本 skill 不额外存储
